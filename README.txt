◆プロジェクトのバックアップ手順
　開発作業をすると、保存するには不要なデータが作成され、データサイズが大きくなります。
　プロジェクトをローカル環境でバックアップを作成する際に不要なデータは削除しましょう

　・ソリューションのルートフォルダ ( DxeUtfSolution*** )内の２つのフォルダを削除します
　　.vs フォルダ
　　x64 フォルダ

　　※ .vs フォルダは隠しフォルダになっている可能性があるため、見つからない場合はフォルダオプションで表示しましょう


　・プロジェクトのルートフォルダ ( 例 000_GameDevelop ) 内の１つのフォルダを削除します
　　x64 フォルダ

　不要なデータを削除した後、ソリューションフォルダを zip 圧縮します
